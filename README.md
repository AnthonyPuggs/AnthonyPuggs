I'm an economics and finance graduate based in Australia, with experience in government programme assessment and a strong interest in macroeconomics, econometrics, public policy, and economic data infrastructure.

I like building tools that make Australian economic data easier to access, analyse, and explain.

- Building tools for Australian economic data from the RBA and ABS

- Exploring HANK/DSGE modelling, causal inference, and applied policy analysis
- Creating interactive economics resources for the web
